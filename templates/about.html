About this project

A Final Year Project on uterine health monitoring using AI. End-to-end pipeline taking raw EUG signals from abdominal electrodes and telling you whether the activity is menstrual — with 0.93 mean fold AUC.

For patients

  • 1-click ESP32 recording from website
  • Real AI chat that answers anything
  • 14 features × 4 phase thresholds
  • Plain-English explanations
  • Cycle predictions + wellness tips

For clinicians

  • Per-contraction predictions
  • 6-check quality scorecard
  • Phase voting matrix
  • Auto-saves .mat + .csv + .txt
  • Printable clinical report

Tech stack

  • Backend: FastAPI · scikit-learn · numpy · scipy
  • Frontend: Vanilla JS · Plotly · Three.js · Canvas
  • Hardware: ESP32-WROOM-32 · 3-channel EUG PCB
  • Signal: 2 kHz · 0.1-3 Hz EUG band
  • AI: GradientBoosting + Ollama LLM (or fallback)
